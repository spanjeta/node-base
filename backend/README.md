# Backend

# node-js
A barebones Node.js app using Express 4.

# Running Locally
Make sure you have Node.js installed.

# Requirements
1) Node 10.0.0(above)
2) Mogodb 4.0(above)

# Common setup
Clone the repo and install the dependencies.

1) git clone REPO
2) cd REPO/backend
3) npm install
4) npm start

Your app should now be running on http://localhost:3000.
